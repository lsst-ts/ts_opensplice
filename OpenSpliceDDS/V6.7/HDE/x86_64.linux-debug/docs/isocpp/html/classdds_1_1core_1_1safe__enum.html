Detailed Description

template<typename def, typename inner = typename def::Type>
class dds::core::safe_enum< def, inner >

safe_enum provides a wrapper for enumerated types in a typesafe manner. safe_enums allow specification of the underlying type, do not implictly convert to integers, and resolve scoping issues.
