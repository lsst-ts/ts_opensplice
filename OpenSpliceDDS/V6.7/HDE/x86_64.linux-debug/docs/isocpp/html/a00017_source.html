13 Throughput {#streams_throughput}
14 ----------
15 - The Streams Throughput example measures data throughput in bytes per second using
16 the Streams API to reduce overhead and maximuse the transfer rate. The publisher allows
17 you to specify a payload size in bytes as well as allowing you to specify when to flush
18 the stream after either a specified number of samples has been sent or after a time out
19 duration. The publisher will continue to send data forever unless a program time out is
20 specified. The subscriber will receive data and output the total amount received and the
21 data rate in bytes per second. It will also indicate if any samples were received out of
22 order. The maximum samples per read can be varied, for best perormance this should match
23 the flush max samples. A maximum number of cycles can be specified and once this has been
24 reached the subscriber will terminate and output totals and averages.

26 Building in a POSIX environment with make {#isocpp_streams_examples_posixbuild}

28 
29 In the following sections `$OSPL_HOME` identifies the installation directory of

33 Building a single example
34 -------------------------
35 
36 Change directory to the example source directory and `make`. E.g.:
37 
38  $ cd $OSPL_HOME/examples/streams/Throughput/isocpp
39  $ make
40 
41 Building all examples
42 ---------------------
43 
44 The top-level makefile `Makefile.DCPS_ISO_CPlusPlus` can be used to build all
45 the examples.
46 
47  $ cd $OSPL_HOME/examples
48  $ make -f Makefile.DCPS_ISO_CPlusPlus

50 Building on Windows with Visual Studio {#isocpp_streams_examples_winbuild}
51 ======================================
52 
53 A link is installed on the Windows Start Menu in the *Examples* Folder under

55 the OpenSplice HDE installation with Windows Explorer or the Visual Studio
56 `file -> open` dialogue and open the solution file `DCPS_ISO_CPlusPlus.sln`.

58 Running examples {#isocpp_streams_examples_running}
59 ================
60 
61 The environment property `OSPL_URI` can set to indicate the location of a local
62 domain configuration file. If this variable is not set, or is set to point to a
63 file that specifies a _shared memory_ domain configuration, the command
64 `ospl start` must be used to start the domain before an example can be run.
65 
66 The default configuration file installed by the distribution uses _single process_
67 (i.e. heap memory) mode so this is not required.
68 
69 POSIX:
70 ------
71 
72 In the following examples `$OSPL_HOME` identifies the installation directory of
73 OpenSplice. *Note* it is not however required that this be exported as a variable
74 in the shell environment.
75 
76 A script `release.com` is provided. It exports a value of `OSPL_URI` to indicate
77 the file `$OSPL_HOME/etc/config/ospl.xml` and also makes the OpenSplice libaries
78 available on the library load path.
79 
80 To run an example in _single process_ mode:
81 
82  $ cd $OSPL_HOME
83  $ ./release.com
84  $ cd examples/streams/Throughput/isocpp
85  $ subscriber &
86  $ publisher
87 
88 To run an exmaple in _shared memory_ mode you might do:
89 
90  $ cd $OSPL_HOME
91  $ ./release.com
92  $ OSPL_URI=file://$OSPL_HOME/etc/config/ospl_shmem_ddsi.xml
93  $ export OSPL_URI
94  $ cd examples/streams/Throughput/isocpp
95  $ ospl start
96  $ subscriber &
97  $ publisher
98  $ ospl stop

100 Windows:
101 --------
102 
103 In the following examples `%%OSPL_HOME%` identifies the installation directory of
104 OpenSplice. *Note* it is not however required that this be set as a variable
105 in the console environment.
106 
107 A script `release.bat` is provided. It exports a value of `OSPL_URI` to indicate
108 the file `%%OSPL_HOME%\etc\config\ospl.xml` and also makes the OpenSplice
109 executables and dynamic-link libraries available on the `PATH`.
110 
111 A shortcut within the OpenSplice HDE folder on the Start Menu: **OpenSplice DDS
112 command prompt** creates a `cmd.exe` console session with this environment ready
113 set.
114 
115 To run an example in _single process_ mode:
116 
117  "<<< OpenSplice HDE Release V6.3.0 For x86_64.win64, Date 2013-03-27 >>>"
118  C:\ospl\V6.3.0\HDE\x86_64.win64>cd examples\streams\Throughput\isocpp
119  C:\ospl\V6.3.0\HDE\x86_64.win64\examples\streams\Throughput\isocpp>start subscriber
120  C:\ospl\V6.3.0\HDE\x86_64.win64\examples\streams\Throughput\isocpp>start publisher
121

128  C:\ospl\V6.3.0\HDE\x86_64.win64\examples\streams\Throughput\isocpp>start subscriber
129  C:\ospl\V6.3.0\HDE\x86_64.win64\examples\streams\Throughput\isocpp>start publisher
130  C:\ospl\V6.3.0\HDE\x86_64.win64\examples\streams\Throughput\isocpp>ospl stop
