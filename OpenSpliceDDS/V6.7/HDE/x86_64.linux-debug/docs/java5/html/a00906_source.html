1 Java 5 DCPS API {#mainpage}
2 ====================
3 
4 [TOC]
5 
6 Java 5 Language PSM for DDS is the latest DDS DCPS API for the Java Language. It is more user friendly
7 and simpler to use.
8 
9 Getting started {#mainpage_gettingstarted}
10 ===============
11 
12 If you are new to the Java 5 PSM we would recommend that you start by looking at the example
13 code provided in your OpenSplice installation directory as well as the documentation
14 in the sections below.
15 
16 A summary of each example, along with build/run instructions can be found on the
17 \subpage java5_examples page.
18 
19 The OMG Java 5 specification also gives an overview of the general design of Java 5 and
20 can be found at the following link: http://www.omg.org/spec/DDS-Java/
21 
22 In order to build and or run applications that use Java 5 the library `dcpssaj5.jar` for standalone Java or `dcpscj5.jar`
23 for CORBA Java need to be in the classPath. In case of doubt use the `dcpssaj5.jar` for standalone Java.
24 These libraries can be found inside the `$OSPL_HOME/jar` directory.

26 API documentation {#mainpage_api}
27 =================
28 
29 The namespaces below contain all of the major functionality of the Java 5 API.
30 
31 - org::omg::dds
32 
33  + org::omg.dds.domain
34  + The org::omg::dds::domain namespace provides facilities for creating a DomainParticipant
35  that establishes a virtual network in which an application and it's entities
36  can operate.
37  + org::omg::dds::topic
38  + The org::omg::dds::topic namespace provides facilities for the creation of Topics which
39  can be published or subscribed to, as well as providing more advanced filtering
40  through ContentFilteredTopics.
41  + org::omg::dds::pub
42  + The org::omg::dds::pub namespace provides facilities for writing samples based on a Topic.
43  + org::omg::dds::sub
44  + The org::omg::dds::sub namespace provides facilities for reading samples based on a Topic,
45  as well as providing the ability to query a read.
46  + org::omg::dds::core
47  + The org::omg::dds::core namespace provides core facilities such as QoS policies and the
48  ability to wait on particular conditions using a WaitSet.

52 Java 5 Source:
53 --------
54 
55 A zipped version of the java5 source code can be found inside the `$OSPL_HOME/lib` directory i.e. `dcpssaj5-src.jar` or `dcpscj5-src.jar`.
56 
57 To use this source jar inside Eclipse do the following:
58 
59 1. Right click over the Project -> Build path -> Configure Build Path
60 2. In the new window, go to the "Libraries" tab.
61 3. Select the library `dcpssaj5.jar` or `dcpscj5.jar` and expand it.
62 4. There are 4 child options. Select "Source attachment" and click the "Edit" button on the right.
63 5. Now the `dcpssaj5-src.jar` or `dcpscj5-src.jar` can be added.
64 
65 JavaDoc:
66 --------
67 
68 A zipped version of the JavaDoc documentation for java5 can be found inside the `$OSPL_HOME/docs/java5` directory i.e. `dcpssaj5-doc.jar` or `dcpscj5-doc.jar`.
69 
70 To use this jar inside Eclipse do the following:
71 
72 1. Right click over the Project -> Build path -> Configure Build Path
73 2. In the new window, go to the "Libraries" tab.
74 3. Select the library `dcpssaj5.jar` or `dcpscj5.jar` and expand it.
75 4. There are 4 child options. Select "Javadoc location" and click the "Edit" button on the right.
76 5. Now the `dcpssaj5-doc.jar` or `dcpscj5-doc.jar` can be added. press the "Validate" button to check everything is ok.
